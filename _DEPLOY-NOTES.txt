# Finch Theory Workplace — Deploy Notes
# workplace.finchtheory.com | GitHub: AetasAi/FinchTheoryWorkplace

============================================================
STANDARD DEPLOY SEQUENCE
============================================================

1. Make changes locally
2. Commit in GitHub Desktop with a clear message
3. Push to main
4. Wait 2-3 minutes for GitHub Pages to build
5. ** PURGE CLOUDFLARE CACHE **
   → Cloudflare dashboard → workplace.finchtheory.com zone
   → Caching → Configuration → Purge Everything
6. Verify at https://workplace.finchtheory.com

============================================================
ADDING NEW INSIGHT ARTICLES
============================================================

1. Build article HTML matching existing article template
2. Drop into insights/
3. Update insights.html hub to include the new article
4. Run: py generate-sitemap.py
5. Commit and push
6. Purge Cloudflare cache
7. GSC → URL Inspection → Request Indexing for new URL
